# Automated segmentation of glioblastoma tumors from MRI scans
Develop a deep learning medical segmentation model using the nnUNet framework to perform automated tumor sub-region segmentation of Glioblastoma multi-parametric MRI scans. Published paper in JEI: https://doi.org/10.59720/23-265
